دورة Python للمبتدئين من الصفر

5 دروس مصورة مع صور توضيحية وتمارين عملية

الدرس 1: ما هي المتغيرات؟

المتغيرات هي صناديق تخزن البيانات في الذاكرة. مثال: تخيل صندوقًا يحمل اسمك أو رقمك.

name = "أحمد"
age = 25
print(name)  # يطبع: أحمد

تمرين: أنشئ متغيرًا لاسمك واطبعه

جرب الكود وشغله. إذا نجح، هتظهر رسالتك!

الدرس 2: الدوال (Functions)

الدوال هي كتل كود تُستدعى مرات متعددة. مثال: دالة ترحيب.

def greet(name):
    print("مرحبا، " + name + "!")

greet("أحمد")  # يطبع: مرحبا، أحمد!

تمرين: أنشئ دالة لحساب المجموع

اكتب دالة تأخذ رقمين وتطبع مجموعها.

الدرس 3: الحلقات (For و While)

الحلقات تكرر الكود. مثال: طباعة أرقام من 1 إلى 5.

for i in range(1, 6):
    print(i)  # يطبع: 1 2 3 4 5

# أو while
count = 0
while count < 5:
    print(count)
    count += 1

تمرين: حلقة لطباعة الأعداد الزوجية

استخدم for لطباعة 2, 4, 6, 8, 10.

الدرس 4: الشروط (If-Else)

الشروط تتخذ قرارات. مثال: التحقق إذا كان العمر أكبر من 18.

age = 20
if age >= 18:
    print("أنت بالغ")
else:
    print("أنت قاصر")

تمرين: برنامج يحدد الدرجة (A/B/C)

أدخل درجة واطبع التقييم.

الدرس 5: مشروع أول: حاسبة بسيطة

اجمع كل شيء في مشروع: حاسبة تأخذ أرقام وتعمل عمليات.

def calculator(a, b, op):
    if op == '+':
        return a + b
    elif op == '-':
        return a - b
    # أضف المزيد...

print(calculator(10, 5, '+'))  # يطبع: 15

تمرين: أضف عملية الضرب والقسمة

وسع الحاسبة واختبرها.
